Refget version 2 - reference sequence retrieval implementation

The specification

Refget enables access to reference sequences using an identifier derived from the sequence itself or a unique identifier. More information can be found from the samtools hosted refget specification page. In addition an OpenAPI 3.0 YAML definition is available. Or you can look at our refget landing page.
